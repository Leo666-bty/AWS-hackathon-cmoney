The signal

ChatGPT 使用率低，不一定是模型不夠好

使用者必須先離開看盤流程、想一個問題，再組織成 Prompt。

每多一步，AI 就離投資現場更遠一步。
打開 Chat↓想問題↓輸入↓等待答案

真正的突破不是讓對話更長，而是讓使用者根本不需要開口。

使用者確認才是真實持股；AI 候選永遠不會直接寫入 Portfolio。

AI done differently

AI 生成可操作介面與建議追問，不再等空白 Prompt

{
  "type": "signal_divergence",
  "stock": "2382",
  "evidence": [
    "法人 20 日 -60,265 張",
    "社群明確多空 93.9% 偏多"
  ],
  "actions": [
    "是我的持股",
    "只是觀察",
    "不相關"
  ]
}
Generative Action Card

Bedrock 負責解釋、卡片與 contextual follow-up；Python 負責算數、證據與排序。

模型失敗時，固定模板仍可完成整個流程。

Data integrity

市場資料與個人資料分開，才不會把推測說成事實

官方 CSV市場、法人、基本面、社群彙總

支撐洞察，不包含個人持股。

LEO 一鍵回饋持有／觀察／不相關

唯一能寫入 Portfolio 的來源。

刻意不知道股數、成本、券商、總資產

三天版本不索取、不推測。

Business outcome

我們優化的是首檔持股確認，不是聊天次數

個股開啟→證據查看→首檔持股確認→Concern 回饋→Portfolio 查看

每一步都形成可同步事件；Dashboard 驗證漏斗，而不是只展示漂亮畫面。

Three-day build

兩人三天，只做一條可信的完整閉環

Day 1閉環先跑通

React、FastAPI、PostgreSQL、event outbox

Day 2AI 與量測

Moment Detector、Bedrock、Concern、Dashboard

Day 3凍結與交付

部署、Demo、簡報、影片與 fallback

只訓練市場時刻偵測；不做持股／焦慮預測、RAG、聊天 Agent 或完整推播。
